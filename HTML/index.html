Garage Virtuel est un site statique pédagogique présentant des collections de voitures, des mini-projets et des démonstrations JavaScript sans dépendances externes.

À propos du projet

Ce projet a pour objectif de proposer un terrain d'apprentissage pour le développement web : structure HTML simple, styles CSS clairs et logique JavaScript côté client pour la gestion d'un "garage" (CRUD minimal, persistance via localStorage).

Objectifs

  • Fournir un exemple clair d'application front-end sans framework.
  • Montrer des patterns simples de manipulation du DOM et de persistance locale.
  • Servir de base pour des ateliers ou mini-projets éducatifs.

Fonctionnalités clés

  • Affichage et filtrage des véhicules depuis des données locales.
  • Ajout / suppression d'éléments via des formulaires modulaires.
  • Persistance avec localStorage et initialisation automatique si vide.

Stack & structure

HTML + CSS + JavaScript (vanilla). Principaux fichiers :

  • Styles : Styles/style.css, Styles/garage.css
  • Logique : Scripts/core/garage.js, Scripts/ui/garage-ui.js, Scripts/ui/forms.js
  • Données exemples : Data/cars.json

Évolution future

Perspectives d'évolution pour le projet :

  • Ajout d'une base de données pour stocker durablement les véhicules (ex. SQLite, PostgreSQL, ou une solution cloud).
  • Gestion d'utilisateurs et authentification (inscription, connexion, rôles et permissions).
  • Développement d'une API backend pour synchronisation multi-utilisateurs et recherches avancées.
  • Import/export, sauvegardes et migrations de données pour fiabilité et portabilité.
  • Améliorations UX : tableaux de bord, filtres avancés et notifications en temps réel.

Contribuer

Pour contribuer, créez une branche, modifiez les fichiers et proposez une pull request. Respectez la simplicité du dépôt : pas de dépendances externes ni de bundler.

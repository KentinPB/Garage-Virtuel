🎯 Objectif du jour

Comprendre la logique Flexbox : conteneur et éléments flexibles.

Apprendre à aligner et répartir les blocs horizontalement et verticalement.

Créer une mise en page en colonnes et en cartes.

Préparer la base pour le responsive design (adaptation mobile).

🧩 Étape 1 – L’idée de base de Flexbox

Flexbox (Flexible Box Layout) est un système qui gère l’espace et l’alignement entre les éléments d’un conteneur.

Tu as déjà utilisé :

display: flex;
justify-content: center;
gap: 70px;


Eh bien, ça, c’était le tout début de Flexbox.

L’idée clé :

Le conteneur flex (parent) contrôle l’agencement.

Les éléments flexibles (enfants) s’ajustent automatiquement.

⚙️ Étape 2 – Les propriétés essentielles
1. display: flex;

Active le mode flex sur le conteneur.

2. flex-direction

Définit la direction des éléments :

flex-direction: row; /* (par défaut) alignés horizontalement */
flex-direction: column; /* empilés verticalement */

3. justify-content

Gère l’alignement horizontal (axe principal) :

flex-start → collés à gauche

center → centrés

space-between → espace égal entre les éléments

space-around → espace égal autour de chaque élément

4. align-items

Gère l’alignement vertical (axe secondaire) :

flex-start → en haut

center → centrés

flex-end → en bas

5. flex-wrap

Permet le retour à la ligne automatique :

flex-wrap: wrap;


→ Si l’écran est trop petit, les éléments passent à la ligne suivante.

6. gap

Crée un espacement uniforme entre les éléments (tu l’utilises déjà très bien).

🧱 Étape 3 – Mise en pratique

Crée un fichier nommé garage.css dans ton dossier styles/ (tu le lieras à projets.html).

Puis, ajoute ceci :

/* Conteneur principal */
.garage-container {
  display: flex;
  flex-wrap: wrap;
  justify-content: center;
  align-items: flex-start;
  gap: 30px;
  margin: 40px auto;
  max-width: 1200px;
}

/* Chaque "carte" de voiture */
.voiture {
  background-color: #252525;
  color: #f0f0f0;
  border-radius: 10px;
  padding: 20px;
  width: 300px;
  box-shadow: 0 4px 8px rgba(0,0,0,0.2);
  transition: transform 0.3s;
}

.voiture:hover {
  transform: scale(1.03);
}

/* Image dans la carte */
.voiture img {
  width: 100%;
  border-radius: 10px;
}

.voiture h3 {
  text-align: center;
  color: #e63946;
}

.voiture p {
  text-align: justify;
  font-size: 14px;
}
